Install GitExtensions from google: http://code.google.com/p/gitextensions/
Install plug-in from codeplex: http://gitscc.codeplex.com/

Select Plug-in provider as 'Git Source Control Provider' in VS - > Tools -> Options -> Source Control -> Plug-in Slection
